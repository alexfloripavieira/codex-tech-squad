# Usage

## Full Delivery

```text
Use codex-tech-squad:squad to form the necessary Codex team and implement this feature with tests.
```

## Discovery Only

```text
Use codex-tech-squad:discovery to analyze this repository and produce an implementation blueprint.
```

## Implementation From Existing Plan

```text
Use codex-tech-squad:implement with subagents to execute this plan.
```

## Bug Fix

```text
Use codex-tech-squad:bug-fix with a small team to investigate, patch, test, and verify this bug.
```

## Hotfix

```text
Use codex-tech-squad:hotfix for this production issue. Keep the patch minimal.
```

## PR Review

```text
Use codex-tech-squad:pr-review to review PR 123 with relevant specialist reviewers.
```

## Important

To authorize subagents, include explicit language such as:

- "use a team"
- "use subagents"
- "use parallel agents"
- "form the necessary squad"

Without explicit team language, Codex should usually run the workflow locally.
